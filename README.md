📊 Amazon Sales Dashboard | Excel Data Analysis Project

This project presents an interactive Amazon Sales Analysis Dashboard built in Microsoft Excel to analyze e-commerce performance across sales, profit, customers, and products. The dashboard converts raw sales data into meaningful business insights using KPI tracking, pivot tables, and visual analytics.

🚀 Project Highlights
• Analyzed sales performance across product categories and customer segments
• Identified top customers and top 10 products by profit
• Visualized profit distribution across countries
• Built KPI cards for Total Sales, Profit, Quantity, and Average Order Value (AOV)
• Implemented interactive filters for Year, Quarter, and Segment

🛠 Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Dashboard Design
Data Visualization

📈 Key Insights
• Consumer segment contributes the highest share of sales
• Technology category generates the largest revenue
• Certain subcategories such as Copiers and Phones drive high profitability

🎯 Objective
The goal of this project is to demonstrate practical data analysis and dashboard development skills by transforming raw e-commerce data into clear insights for business decision-making.
